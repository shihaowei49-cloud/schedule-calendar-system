# 📅 日程安排系统

一个简洁优雅的个人日历管理工具，帮助你规划每一天。

## ✨ 功能特点

- 📅 月度日历视图
- ⏰ 时间段管理
- ✅ 任务清单
- 😊 心情记录
- 🌤️ 天气标记
- 📖 详细笔记
- 🎯 目标追踪
- 🏷️ 标签分类

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start
```

访问 http://localhost:3000

## 📦 部署到Zeabur

1. 推送代码到GitHub
2. 在Zeabur导入仓库
3. 自动部署完成

## 📝 使用说明

1. 点击日历上的日期查看或编辑当天日程
2. 填写标题、心情、天气等基本信息
3. 添加时间段和活动
4. 创建任务清单并标记优先级
5. 记录详细笔记和总结
6. 保存后在日历上查看

## 🛠️ 技术栈

- HTML5 + CSS3 + JavaScript
- Node.js + Express
- LocalStorage 数据存储
